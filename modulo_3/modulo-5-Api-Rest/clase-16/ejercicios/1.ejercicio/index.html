Tabla Resumen de Endpoints

Recursos, métodos soportados y parámetros de filtrado.

GET: Obtener POST: Crear PUT: Editar DELETE: Eliminar
Recurso / Endpoint Métodos HTTP Soportados Búsqueda y Filtros Observaciones Extra
Products
/products
/products/:id
GET POST PUT DELETE
  • title (Nombre)
  • price (Precio exacto)
  • price_min / max (Rango de precios)
  • categoryId o categorySlug
  • slug (Búsqueda semántica)
  • limit / offset (Paginado)
 Permite editar y eliminar productos existentes por ID.
Extra: Permite lógica para obtener productos relacionados.
Users
/users
/users/:id
GET POST PUT

Paginado: Sí

limit offset

Verifica disponibilidad de email (lógica interna).

 Gestión de usuarios. Importante para administrar perfiles.
Categories
/categories
/categories/:id
GET POST PUT DELETE
Paginado: Sí (limit/offset).
Permite obtener productos relacionados a una categoría específica: /categories/:id/products
Servicios Adicionales
Authentication
/auth/login /auth/profile
GET POST
Requiere body con email/password o Header con Token. Manejo de JWT (Tokens de acceso y refresco).
Files
/files/upload /files/:name
GET POST
Una vez cargados los archivos, se pueden referenciar en otros recursos. Subida y recuperación de imágenes/archivos.
Locations
/locations 		GET POST Filtrado Geoespacial:
  • radius (Radio en km)
  • lat (Latitud)
  • lng (Longitud)
  • limit / offset (Paginado)
 Recurso para gestión de ubicaciones geográficas.
