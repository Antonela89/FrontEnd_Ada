Preguntas Frecuentes (FAQ)

¿Qué es JavaScript?

JavaScript es un lenguaje de programación interpretado, multiparadigma y de alto nivel. Se utiliza principalmente para crear interactividad en sitios web, pero también se ha extendido a otras áreas como el desarrollo de aplicaciones móviles, de escritorio y servidores (Node.js).

¿Cuál es la diferencia entre `let`, `const` y `var`?

var tiene un alcance de función y permite el "hoisting". let y const tienen un alcance de bloque (block scope) y son introducidos en ES6. La diferencia principal es que let permite reasignar su valor, mientras que const declara una constante que no puede ser reasignada después de su inicialización.

¿Qué es el DOM?

El DOM (Document Object Model) es una interfaz de programación para documentos HTML, XML y SVG. Proporciona una representación estructurada del documento como un árbol y define cómo los programas pueden acceder y cambiar el contenido, estructura y estilo de un documento.

¿Qué es un evento en JavaScript?

Un evento es una acción o suceso que ocurre en el sistema o en el navegador web que JavaScript puede detectar y a la que puede reaccionar. Ejemplos comunes incluyen clics de ratón, pulsaciones de teclas, carga de una página, envío de formularios, etc.

¿Para qué se usa `preventDefault()`?

El método `preventDefault()` se utiliza en el contexto de eventos para detener la acción predeterminada que el navegador normalmente realizaría para ese evento. Por ejemplo, evitar que un formulario se envíe o que un enlace redirija a otra página.

¿Cómo se selecciona un elemento HTML con JavaScript?

Se pueden seleccionar elementos usando métodos como `document.getElementById('idDelElemento')`, `document.querySelector('.claseDelElemento')` (para el primer elemento que coincida) o `document.querySelectorAll('etiqueta')` (para todos los elementos que coincidan).
