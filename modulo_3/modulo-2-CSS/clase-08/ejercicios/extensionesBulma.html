Accordion

Crea paneles de contenido colapsables, ideales para FAQs. Necesita el CSS y JS de `bulma-accordion`.

¿Cómo funciona?

El JS adjunto gestiona el estado de apertura y cierre.

¿Es personalizable?

Sí, puedes cambiar colores y tamaños con las clases de Bulma.

Muestra eventos en orden cronológico. Solo necesita el CSS de `bulma-timeline`.
