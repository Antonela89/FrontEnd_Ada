Comparativa de Frameworks CSS

Construyendo la misma Tarjeta de Producto con Bootstrap, Bulma y Tailwind

Paso 1: Estructura Base (La Card)

Bootstrap

Usa el componente .card. Rápido y directo.
Todo está predefinido.

Zapatillas Deportivas

Unas zapatillas cómodas y con estilo.

<div class="card bg-dark text-white">
    <img src="..." class="card-img-top" alt="Producto">
    <div class="card-body">
        <h5 class="card-title">....título....</h5>
        <p class="card-text small">...descripción...</p>
    </div>
</div>

Bulma

Similar a Bootstrap con el componente .card.
Estructura semántica clara.

Zapatillas Deportivas

Unas zapatillas cómodas y con estilo.

        </figure>
    </div>

        <p class="title is-5 mb-2 has-text-light">...</p>
        <p class="subtitle is-6 has-text-light">...</p>
    </div>
</div>

Se construye desde cero con utilidades.
Control total sobre cada detalle.

Paso 2: Badge y Selectores (Colores y Tallas)

Bootstrap

El badge es fácil con .position-absolute. Para los selectores, .btn-group es ideal. Los círculos de color requieren estilo manual.

OFERTA
Zapatillas Deportivas
<div class="card ... position-relative">
    <span class="badge text-bg-danger position-absolute top-0 start-0 m-2">OFERTA</span>
    <!-- ... img ... -->
    <div class="card-body">
        <h5>...</h5>
        <!-- Círculos (con estilo inline o CSS) -->
        <div class="d-flex gap-2 my-2">
            <div style="width:20px; ... background-color: red;"></div>
            <div style="width:20px; ... background-color: blue;"></div>
        </div>
        <!-- Selector de Talla -->
        <div class="btn-group w-100 mt-2">
            <button class="btn btn-outline-secondary">S</button>
            <button class="btn btn-outline-secondary">M</button>
        </div>
    </div>
</div>

Bulma

El badge necesita un helper de CSS para posicionarlo. .buttons.has-addons es perfecto para los botones de talla.

Zapatillas Deportivas

<div class="card ...">
    <div class="card-image">
        <figure class="image is-4by3 is-relative">
            <!-- Requiere un helper CSS para 'is-relative' y 'is-absolute' -->
            <span class="tag is-danger is-absolute">OFERTA</span>
            <img src="..." alt="Producto">
        </figure>
    </div>
    <div class="card-content">
        <p>...</p>
        <!-- Círculos (con estilo inline o CSS) -->
        <div class="is-flex" style="gap: 0.5rem;">
            <div style="width:20px; ... background-color: red;"></div>
            <div style="width:20px; ... background-color: blue;"></div>
        </div>
        <!-- Selector de Talla -->
        <div class="buttons has-addons is-fullwidth">
            <button class="button is-outlined is-light is-fullwidth">S</button>
            <button class="button is-outlined is-light is-fullwidth">M</button>
        </div>
    </div>
</div>

Tailwind CSS

Aquí brilla. El badge, los círculos (incluido el estado focus) y los botones se construyen con utilidades sin una línea de CSS.
